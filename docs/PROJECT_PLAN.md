# InkIn - 项目规划书

> **入画** — 让照片走进艺术的世界

---

## 一、项目概述

### 1.1 项目名称
**InkIn**（入画）

### 1.2 项目定位
一款本地运行的 Web 应用，用户上传照片后，通过调用大模型 API 自动生成漫画/艺术风格的图片。支持任意兼容 OpenAI 格式的模型提供商，用户可自由配置 API Key、接口地址和模型 ID。

### 1.3 核心理念
- **入画**：从现实（照片）进入艺术（漫画）的世界
- **开放**：不绑定特定平台，支持任意模型提供商
- **简洁**：专注核心功能，用户体验优先
- **本地**：数据不离开用户电脑，隐私安全

### 1.4 目标用户
- 想要把照片变成漫画头像的普通用户
- 需要漫画风格素材的设计师/内容创作者
- 想要体验不同 AI 模型效果的技术爱好者

### 3.3 API 适配设计

采用 **OpenAI Chat Completions 格式** 作为统一接口标准：

```json
{
  "model": "用户配置的模型ID",
  "messages": [
    {
      "role": "user",
      "content": [
        {
          "type": "text",
          "text": "将这张照片转换为漫画风格"
        },
        {
          "type": "image_url",
          "image_url": {
            "url": "data:image/jpeg;base64,{图片base64}"
          }
        }
      ]
    }
  ],
  "max_tokens": 4096
}
```

**兼容性说明**：
- ✅ OpenAI（GPT-4o, GPT-4 Vision）
- ✅ 豆包（字节跳动）
- ✅ 通义千问（阿里）
- ✅ 智谱 AI
- ✅ 其他兼容 OpenAI 格式的第三方服务

## 五、核心功能设计

### 5.1 图片上传模块

**功能**：
- 支持拖拽上传
- 支持点击选择文件
- 支持粘贴图片
- 实时预览原图
- 文件格式校验（JPG/PNG/WEBP）
- 文件大小限制（默认 10MB）

**交互流程**：
```
用户拖拽/选择图片
    ↓
前端校验格式和大小
    ↓
显示原图预览
    ↓
用户点击"生成漫画"
    ↓
图片转 base64 发送到后端
```

### 5.2 API 配置模块

**配置项**：
| 字段 | 说明 | 示例 |
|------|------|------|
| API Key | 访问密钥 | sk-xxx... |
| Base URL | 接口地址 | https://api.openai.com/v1 |
| Model ID | 模型名称 | gpt-4o |
| Prompt | 提示词 | 将照片转换为漫画风格 |
| Timeout | 超时时间 | 120 秒 |

**配置管理**：
- 支持保存多组配置
- 支持快速切换配置
- 配置持久化到本地文件
- 敏感信息（API Key）本地存储，不上传

### 5.3 风格预设模块（中期）

内置多种风格预设：

| 风格 | Prompt 示例 |
|------|-------------|
| 日系动漫 | 将照片转换为日本动漫风格，大眼睛，精致线条 |
| 美式漫画 | 将照片转换为美式漫画风格，粗线条，高对比度 |
| 水彩画 | 将照片转换为水彩画风格，柔和色彩，晕染效果 |
| 油画 | 将照片转换为油画风格，厚涂笔触，丰富质感 |
| 像素风 | 将照片转换为像素艺术风格，复古游戏感 |
| 赛博朋克 | 将照片转换为赛博朋克风格，霓虹灯，未来感 |
| 水墨画 | 将照片转换为中国水墨画风格，黑白为主，写意 |

### 5.4 结果展示模块

**功能**：
- 显示生成的漫画图片
- 原图/漫画对比展示
- 一键下载漫画图片
- 调整图片大小/缩放
- 复制图片到剪贴板

### 5.5 历史记录模块（中期）

**功能**：
- 保存处理历史（原图 + 漫画）
- 记录使用的配置和提示词
- 支持重新生成
- 支持删除记录
- 历史记录导出

## 九、风险与挑战

### 9.1 技术风险

| 风险 | 影响 | 应对方案 |
|------|------|----------|
| API 返回格式不一致 | 解析失败 | 统一适配层，处理多种返回格式 |
| 大图片处理慢 | 用户体验差 | 前端压缩图片，设置合理超时 |
| API 限流/报错 | 功能不可用 | 友好错误提示，支持重试 |
| 跨域问题 | 前后端不通 | Flask-CORS 配置 |

### 9.2 产品风险

| 风险 | 影响 | 应对方案 |
|------|------|----------|
| 生成效果不理想 | 用户不满意 | 支持调整提示词，多风格选择 |
| API 成本高 | 用户负担重 | 显示 token 用量，支持低成本模型 |
| 隐私担忧 | 用户不信任 | 强调本地运行，数据不上传云端 |

## 十二、总结

**InkIn（入画）** 是一个简洁而强大的照片转漫画工具，核心优势在于：

1. **开放兼容**：不绑定特定平台，支持任意 OpenAI 格式 API
